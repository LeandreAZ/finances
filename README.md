# Finance

## About
During management classes, we had to create two websites simulating financial calculations.
The first one calculates the interest of a bank account based on the interest rate, the duration, and the different account transactions (deposits and withdrawals).
The second one generates a loan amortization schedule based on the borrowed amount, the interest rate, the duration, and the repayment frequency (annual, semi-annual, etc.).
